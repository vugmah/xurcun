# Xurcun

Premium restaurant and lounge website with admin panel, QR menu, and full CMS.

## Tech Stack

- **Frontend:** React + Vite + TypeScript + Tailwind CSS
- **Backend:** Hono + tRPC + Drizzle ORM
- **Database:** MySQL (Railway)
- **Deployment:** Railway

## Features

- 🍽️ Multilingual menu (AZ/RU/EN) with QR code generation
- 📸 Photo management and assignment system
- 📧 Contact form with admin inbox
- 🔍 SEO management per page
- 📊 Google Ads integration (inactive by default)
- 🎯 Tracking support (GA4, GTM, Meta Pixel)
- 🤖 AI Auditor & Insights
- 🖨️ Print preview & PDF/PPTX export
- 🏷️ Badge recommendations system
- 🍹 Popup campaign management

## Scripts

```bash
npm run dev        # Start development server
npm run build      # Build for production
npm run check      # TypeScript type check
npm run start      # Start production server
npm run db:push    # Push schema to database
```
